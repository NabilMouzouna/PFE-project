# @appbase/types

TypeScript definitions shared by the AppBase API and [`@appbase/sdk`](https://www.npmjs.com/package/@appbase/sdk). Consumers rarely install this package directly; it is a dependency of the SDK.

HTTP contract: see [`docs/API-SPEC.md`](https://github.com/NabilMouzouna/NubleCloud-PFE/blob/main/docs/API-SPEC.md) in the main repository.

## Install

```bash
npm install @appbase/types
```

Usually you only need `@appbase/sdk`, which declares a semver range on this package.
